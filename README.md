# Kniha jázd - ZVL SLOVAKIA

Elektronická kniha jázd pre správu vozidlového parku firmy ZVL SLOVAKIA. Zákonná kniha jázd platná od 1.1.2026.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Jazyk:** TypeScript
- **Styling:** Tailwind CSS v4
- **Komponenty:** shadcn/ui
- **Databáza:** Supabase (PostgreSQL)
- **Autentifikácia:** Supabase Auth
- **Grafy:** Recharts
- **Export:** jsPDF, xlsx

## Funkcie

- Dashboard s prehľadom vozidlového parku
- Správa vozidiel (STK, EK, diaľničné známky)
- Správa vodičov
- Evidencia jázd s automatickým číslovaním
- Evidencia tankovania PHM
- Reporty a štatistiky

## Inštalácia

```bash
npm install
```

## Spustenie

```bash
npm run dev
```

Aplikácia beží na [http://localhost:3000](http://localhost:3000)
